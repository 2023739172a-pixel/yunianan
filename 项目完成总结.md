## 🚀 快速启动

### 开发模式

```bash
# 安装依赖（首次）
npm install

# 同时启动前端和后端
npm run dev

# 或者分别启动
npm run server:dev  # 后端（端口 5000）
npm run client:dev  # 前端（端口 5173）
```

### 生产构建

```bash
npm run build
```

### 命令行工具

```bash
# 使用示例数据
npm run cli:sample

# 搜索问题
npm run cli search -- --keyword "你的问题"

# 查看平台
npm run cli platforms
```

## 📱 功能使用说明

### 首页功能
1. **搜索框** - 输入要问的问题
2. **平台选择** - 选择要搜索的 AI 平台
3. **快捷按钮** - 悬浮窗、连续答题、历史记录
4. **体验示例** - 快速查看演示效果

### 悬浮窗使用
1. 点击"悬浮窗"按钮
2. 按住顶部灰色条拖动位置
3. 输入问题，点击"搜索"
4. 查看结果，点击"最小化"隐藏

### 连续答题使用
1. 添加多个题目
2. 点击"开始答题"
3. 系统自动逐个搜索
4. 支持暂停和继续
5. 完成后可在历史记录查看
