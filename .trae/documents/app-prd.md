# AI 答题助手 APP 产品需求文档

## 1. 产品概述

将现有的 AI 答题助手转换为手机 APP 版本，提供更符合移动端习惯的界面和交互体验。

## 2. 核心功能

### 2.1 移动端适配
- 重新设计前端布局，完全适配手机屏幕
- 优化触摸操作，按钮和输入区域更大
- 响应式设计，支持横竖屏

### 2.2 悬浮窗搜题
- 可悬浮在屏幕上的小窗口
- 支持从其他应用复制题目自动搜索
- 快速切换显示/隐藏
- 可调整悬浮窗位置和大小

### 2.3 自动连续答题
- 设置题目数量后自动连续搜索
- 自动记录答题历史
- 支持暂停/继续答题
- 答题进度显示

## 3. 页面设计

### 首页
- 搜索输入框（全屏宽度，大字体）
- 平台选择（卡片式，易点击）
- 快捷按钮区（悬浮窗、连续答题、历史记录）

### 搜索结果页
- 卡片式答案展示
- 左右滑动切换不同平台答案
- 一键复制答案
- 图表可视化（横向滚动）

### 悬浮窗模式
- 迷你输入框
- 快速搜索按钮
- 答案预览弹窗

### 连续答题页
- 题目列表
- 答题进度条
- 暂停/继续/重置按钮
- 批量导出功能

## 4. 技术实现

- PWA (Progressive Web App) 技术
- 支持添加到主屏幕
- 模拟悬浮窗效果（网页实现）
- LocalStorage 存储历史记录
- Service Worker 后台服务
