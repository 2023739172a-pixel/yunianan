## 📋 准备工作

在开始之前，请确保你的电脑已经安装了这两个东西：

### 1️⃣ 安装 Node.js

- 👉 去官网下载：https://nodejs.org/
- 下载 **LTS 版本**（长期支持版，稳定好用）
- 安装时一直点"下一步"就可以了
- 安装完成后，打开终端（Windows 的 cmd/PowerShell，Mac 的终端），输入：
  ```bash
  node -v
  npm -v
  ```
  如果显示了版本号，说明安装成功！

---

## 📦 第一步：下载项目（如果你还没有的话）

如果你是从 GitHub 或其他地方获取的项目代码，先解压到一个文件夹里。

---

## 🔧 第二步：安装依赖

1. 打开终端（Windows: 开始菜单搜索"cmd"或"PowerShell"，Mac: 启动台搜"终端"）

2. 进入项目文件夹（假设你把项目放在桌面上叫 "ai-answer" 的文件夹）：
   ```bash
   # Windows 示例
   cd Desktop\ai-answer
   
   # Mac 示例
   cd Desktop/ai-answer
   ```

3. 安装所有需要的工具（这步可能需要几分钟，耐心等待）：
   ```bash
   npm install
   ```
   
   ✅ 看到很多绿色的对勾或没有报错，就说明安装成功了！

---

## 🎯 第三步：开始使用！

### 方式一：网页版（推荐新手）

1. **启动服务**：
   ```bash
   npm run dev
   ```

2. **打开浏览器**：
   - 看到终端显示类似 "Local: http://localhost:5173" 就可以了
   - 打开浏览器，访问这个地址（通常是 http://localhost:5173）

3. **使用网页**：
   - 在搜索框输入你的问题，比如："什么是人工智能？"
   - 选择要查询的 AI 平台（豆包、千问、元宝）
   - 点击"开始搜索"
   - 查看搜索结果和对比图表！

### 方式二：命令行版（适合喜欢用终端的人）

1. **查看示例数据**：
   ```bash
   npm run cli:sample
   ```

2. **搜索问题**：
   ```bash
   npm run cli search -- --keyword "你想问的问题"
   ```

3. **查看支持的平台**：
   ```bash
   npm run cli platforms
   ```

## 🔧 常见问题解决

### Q: 打开网页显示"无法访问此网站"
**A**: 检查一下终端是否还在运行，不要关掉终端窗口！

### Q: npm install 特别慢
**A**: 可以使用国内镜像加速：
```bash
npm config set registry https://registry.npmmirror.com
```
然后再重新运行 npm install

### Q: 网页加载很慢
**A**: 因为要模拟从多个平台获取答案，这是正常的，请耐心等待！

### Q: 怎么停止服务？
**A**: 在终端窗口按 `Ctrl + C`（Windows）或 `Cmd + C`（Mac）就可以停止了
