
# 🎨 图标文件说明

这是 public 文件夹，用于存放静态资源。

## 需要的图标文件

为了让 PWA 功能更完善，请添加以下图标：

### 必需的图标

1. **icon-192.png** (192x192 像素)
   - 用于 Android 设备
   - 显示在主屏幕

2. **icon-512.png** (512x512 像素)
   - 用于更高分辨率的设备
   - 用于 Splash Screen

### 如何创建图标

**方法一：在线工具**
- 访问 https://favicon.io/ 或类似网站
- 上传你的 Logo
- 生成所需尺寸

**方法二：设计工具**
- 使用 Figma、Photoshop、Sketch 等
- 设计一个简洁的图标
- 建议蓝色主题（#3b82f6）
- 包含"AI"或"答"字样

**方法三：临时方案**
- 在 manifest.json 中注释掉 icons 部分
- 应用仍可使用，只是没有图标

## 其他文件

- **sw.js** - Service Worker 文件（已创建）
- **manifest.json** - PWA 配置文件（已创建）

## 图标设计建议

- 🎯 简洁明了
- 🎨 蓝色主题
- 📱 圆形或圆角矩形
- ✨ 有识别度

## 临时图标

在正式图标制作好之前，可以：
1. 使用系统默认图标
2. 或暂时忽略，不影响功能使用
