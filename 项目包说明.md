## 🚀 使用方法

### 🎯 方法一：双击"一键复制到D盘.bat"（最简单）

#### 步骤：
1. **获取项目包** - 下载或复制到您的电脑
2. **双击运行** - `一键复制到D盘.bat`
3. **等待完成** - 自动复制到 `D:\AI答题助手`
4. **开始构建** - 进入D盘目录，运行构建工具

---

### 🎯 方法二：手动复制

#### 步骤：
1. **复制所有文件** - 选中整个项目包文件夹
2. **粘贴到D盘** - 创建文件夹 `D:\AI答题助手`
3. **粘贴所有内容**
4. **开始构建**

---

## 📋 构建APK的完整流程

### 1️⃣ 检查环境

双击运行：`环境检查工具.bat`

确保已安装：
- Node.js v18+
- Java JDK 17+
- Android SDK

---

### 2️⃣ 开始构建

双击运行：`AI答题助手-APK构建工具.bat`

---

### 3️⃣ 获取APK

构建完成后，APK位于：
```
D:\AI答题助手\android\app\build\outputs\apk\debug\app-debug.apk
```

## ⚠️ 常见问题

### Q: 提示找不到D盘？
A: 请确保您的电脑有D盘，或者修改复制脚本的目标路径

### Q: 复制很慢？
A: 项目包只有约2MB，应该很快。如果很慢，请检查磁盘IO

### Q: 构建时出错？
A: 先运行`环境检查工具.bat`，检查所有依赖是否安装
